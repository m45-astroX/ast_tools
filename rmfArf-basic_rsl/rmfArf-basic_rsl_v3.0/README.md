## 概要

RMF(energy redistribution matrix file), ARF(ancillary response function), expMapを作成するスクリプト。


## 使用方法

    $ bash /path/to/rmfArf-basic_rsl.bash BASENAME evtfile ehkfile expgtifile regionfile centerCalculationMethod

    $1 BASENAME
    $2 evtfile (cleaned evt)
    $3 ehkfile
    $4 exp gti file
    $5 region file
    $6 center calculation method (DEFAULT or SCRIPT)
